# AI Agent Platform V2 产品执行文档（Claude Code）

## 产品定位

将当前 Prompt Builder + Chat Demo 重构为企业级 AI Agent Platform。

### 核心原则

-   双体验模式（Professional / Character）
-   企业级模块化架构
-   Memory、Knowledge、Workflow、Tool Calling 可扩展
-   默认隐藏技术细节，保持拟人体验
-   UI 对标 ChatGPT、Claude、Linear、Raycast、Notion、Vercel

## 功能模块

1.  全新 Dashboard UI（左导航/中聊天/右信息）
2.  Agent 创建中心（身份、性格、Prompt、Goal 等完整配置）
3.  Memory（Short / Long / Profile / Preference / Task / Project）
4.  Knowledge Base（RAG）
5.  多模型 Provider
6.  Tool Calling
7.  Workflow Engine
8.  Prompt Engine
9.  Conversation 管理
10. Context Engine
11. Thinking Mode
12. 文件上传
13. Agent Marketplace
14. 动画系统（Spring Motion）
15. Feature First 架构
16. Engine 解耦
17. MCP / Plugin 预留

## UI规范

-   Glassmorphism
-   大留白
-   Dark / Light / Auto
-   Spring 动画
-   Inter/Geist + MiSans
-   Lucide 图标
-   Command Palette（Ctrl+K）
